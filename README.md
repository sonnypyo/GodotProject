### 1. 프로젝트 소개

- GoDot Engine을 기반으로 제작된 색다른 Chess 게임 제작 프로젝트입니다.

<br>

### 2. 프로젝트 개요

- 제작 기간: 2주
- 개발 도구: Godot Engine
- 개발 언어: GD.Script
